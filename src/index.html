Задание 4. Строки и регулярки

Задачи

  • Задача #1

    Написать скрипт, предлогающий пользователю ввести две строки через запятую. Вывести сообщение true, если вторая строка содержится в первый, в противном случае false, регистр при проверке не учитывать. Для запуска нажмите "Старт"

  • Задача #2

    Пользователь вводит строку, затем число (кол-во символов). Функция усекает строку до кол-ва символов и добавляет многоточие. Для запуска заполните объязательные поля (*) и нажмите "Выполнить"
