Задание 5. Массивы и их методы, Set

Задачи

Выполнение всех заданий запускается из консоли (глобальая переменная window.tasks)

  • Задача #1

    На вход поступает массив, вывести массив, удалив неуникальные значения.
    Запуск из консоли: tasks.task1.run() | tasks.task1.run(array[])

  • Задача #2

    На вход поступает массив, реверсировать значения (подобно методу reverse) метод reverse не использовать.


    Запуск из консоли: tasks.task2.run() | tasks.task1.run(array[])

  • Задача #3

    На вход поступает массив, содержащий массивы, в которых хранится два элемента. Преобразовать массив в объект, где ключами являются нулевой индекс вложенныхых массивов, а значениями являются элементы с индексом один.


    Запуск из консоли: tasks.task3.run() | tasks.task3.run(array[][])

  • Задача #4

    На вход поступает объект, вывести сумму числовых свойств объекта.


    Запуск из консоли: tasks.task4.run() | tasks.task4.run(object{})

  • Задача #5

    На вход поступает массив с числами, вывести среднее арифметическое элементов массива.


    Запуск из консоли: tasks.task5.run() | tasks.task5.run(array[])

  • Задача #6

    Создать функцию-конструктор для объекта "калькулятор", объект должен иметь поле, хранящее текущее значение и методы сложения, вычитания, умножения и деления, принимающие число и манипулирующий свойством значения в соответствии с назначением метода, а так же функцию, сбрасывающую значение в ноль.


    Запуск из консоли: const calc = tasks.task6.run()
    Публичные методы: setVal(num), getVal(), reset(), sum(num), minus(num), div(num), multiply(num)
    Доступна цепочка вызовов, например: calc.sum(10).minus(2).div(4).multiply(3).getVal()

  • Задача #7

    Функция принимает смешанный массив (содержащий значения чисел, строк и объектов), вернуть объект с полями numbers, strings и objects, содержащими массив со значениями, соответствующими названию поля.


    Запуск из консоли: tasks.task7.run() | tasks.task7.run(array[])

  • Задача #8

    Функция принимает массив чисел и два числовых значения, вернуть новый массив, содержащий элементы первого массива, значение которых попадает под диапазон переданных в функцию чисел (второе переданное число может быть больше первого)


    Запуск из консоли: tasks.task8.run() | tasks.task8.run(array[], min?, max?)

  • Задача #9

    Функция принимает две строки. Вывести true, если строки являются анаграммами, в противном случае false


    Запуск из консоли: tasks.task9.run() | tasks.task9.run(string, string)

  • Задача #10

    Создать объект, выводящий в консоль все ключи и значения объекта в формате "ключ: значение" через запятую (считать, что значением ключа объекта не может быть объектом или массивом, содержащими объекты) сама функция в консоль выводиться не должна.


    Запуск из консоли: const viewer = tasks.task10.run(object{})
    Публичные методы: print()

  • Задача #11

    Создать функцию-конструктор для объекта, содержащего методы setProp (установить значение свойства), метод принимает ключь (строка), значение (произвольное) и объект со свойствами writable, configurable, enumerable (разрешение перезаписи свойства, разрешение перечисления свойства и разрешение удаления свойства). Если какое-то из свойств в объекте отсутствует, действие должно быть разрешено


    Запуск из консоли: const mutant = tasks.task11.run()
    Публичные методы: setProp(key, value, opt)
