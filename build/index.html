Задание 7. Прототипы, наследование. ООП, классы

Задачи

Выполнение всех заданий запускается из консоли (глобальая переменная window.tasks)

  • Задача #1

    Реализовать объект animal, с полем клички, методом eat, выводящим сообщение "/*кличка*/ ест" и методом say, выводящим фразу, "неизвестное животное молчит", путём прототипного наследования создать объекты кота, собаки и попугая. Переопределить в них метод say, в зависимости от типа животного. Для кота добавить новый метод hunt, выводящий сообщение "/*кличка*/ охотится". Все перечисленные методы и свойства должны быть защищены от удаления и перезаписи. Методы должны быть неперечисляемыми. Разработать метод rename, для смены клички животного. Новая кличка должна содержать только кирилические символы, пробелы или символ "-". (протатипное наследование)


    Запуск из консоли: const a = new window.tasks.prototype.[[ObjName = Animal | Dog | Cat | Parrot]]

  • Задача #2

    Выполнить то же самое использую функции конструкторы (функуиональное наследование).


    Запуск из консоли: const b = new window.tasks.func.[[FuncName = Animal | Dog | Cat | Parrot]]()

  • Задача #3

    Выполнить то же самое, используя классы (классовое наследование).


    Запуск из консоли: const c = new window.tasks.class.[[ClassName = Animal | Dog | Cat | Parrot]]()
