Задание 5. Массивы и их методы, Set

Задачи

Выполнение всех заданий запускается из консоли (глобальая переменная window.tasks)

  • Задача #1

    Реализовать функцию, принимающую число (индекс в последовательности Фибоначчи), функция должна вернуть значение числа. Использовать рекурсию.


    Запуск из консоли: tasks.task1.run(n)

  • Задача #2

    Модернизировать написанную функцию, добавив кэширование (функция “”запоминает”” входной параметр и вычесленное значение, при следующем вызыве с этим же параметром, функция не вычисляет значение, а возвращает из памяти)


    Запуск из консоли: tasks.task2.run(n)

  • Задача #3

    Разработать рекурсивную функцию, принимающую массив, содержащий массивы из двух элементов, в каждом из которых первый элемент является строкой, а второй строкой, числом, логическим значением, объектом, или таким же массивом. Функция должна преобразовать массив в объект.


    Запуск из консоли: tasks.task3.run(array[])
